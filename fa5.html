Font Awesome Version 5 Table of Icons

This page contains all Font Awesome (FA) Version 5 icons, and a button to copy the HTML code or the Moodle FA Filter code - e.g. [fa-eye]. Simply search for your icon in the search space provided, and then click the button to copy the code you require. This will not include any of the advanced alterations you can make with icons. The tool is intended for you to search, find and copy plain Font Awesome Icons for use in Moodle or web pages.

Search for your icon

Use the search box below to search for your icon's name and the table will automatically reduce your options to match with the name you search. E.g. search for eye and the table will only display the icons with eye in the name.
